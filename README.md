## Intoduction 💬

프로덕션 중인 서비스 내에서 발생하는 오류들을 Monitoro website 에서 실시간으로 모니터링할 수 있습니다.

직접 태그를 설정하고 태그 별 발생한 에러, 에러별 통계 등 부가적인 기능들을 제공하여 보다 쉽게 에러 발생율, 조치 우선 순위, 조치 방안을 세울 수 있습니다.

당신의 소중한 서비스를 항상 지켜보고 안전에 위협이 될만한 부분은 전해드리겠습니다.
